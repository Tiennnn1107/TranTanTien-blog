Fetch API trong JavaScript

December 23, 2025 · 1 phút

Fetch API là gì?#

Fetch API dùng để gửi request HTTP từ trình duyệt.

Ví dụ#

fetch("https://api.example.com/data")
  .then(res => res.json())
  .then(data => console.log(data));
