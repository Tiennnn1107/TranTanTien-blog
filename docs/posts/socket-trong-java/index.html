Socket trong Java là gì?

December 23, 2025 · 1 min · Trần Tấn Tiến

Socket là gì?#

Socket là điểm giao tiếp giữa hai máy tính thông qua mạng. Trong Java, socket giúp client và server trao đổi dữ liệu.

Các lớp chính#

  • Socket: phía client
  • ServerSocket: phía server

Ví dụ đơn giản#

1

Socket socket = new Socket("localhost", 1234);
