## JavaScript là gì?
JavaScript là ngôn ngữ lập trình chạy trên trình duyệt.

## Vai trò
- Tạo website động
- Gửi request
- Xử lý dữ liệu phía client

## Kết hợp với backend
JavaScript thường kết hợp với Java để xây dựng web hoàn chỉnh.
