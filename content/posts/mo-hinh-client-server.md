---
title: "Mô hình Client – Server trong lập trình mạng"
date: 2025-12-21
tags: ["client-server", "network", "java"]
---

Mô hình Client – Server là kiến trúc cơ bản và phổ biến nhất trong các hệ thống mạng hiện nay. Trong mô hình này, Client đóng vai trò gửi yêu cầu, còn Server chịu trách nhiệm tiếp nhận, xử lý và trả kết quả.

Client thường là trình duyệt web, ứng dụng desktop hoặc mobile. Server là nơi chứa dữ liệu và logic xử lý. Ví dụ điển hình là khi người dùng truy cập một website, trình duyệt sẽ gửi yêu cầu đến Server và Server trả về nội dung trang web.

Trong quá trình học Java Socket, mình đã tự xây dựng một chương trình Client gửi chuỗi ký tự đến Server. Server sau khi nhận dữ liệu sẽ xử lý và phản hồi lại. Qua đó, mình hiểu rõ hơn cách các hệ thống giao tiếp với nhau trong thực tế.

Mô hình Client – Server giúp hệ thống dễ quản lý, dễ mở rộng, tuy nhiên cũng đòi hỏi Server phải xử lý tốt nhiều kết nối đồng thời để tránh quá tải.
