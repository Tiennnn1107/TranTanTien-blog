## Client – Server là gì?
Client gửi yêu cầu, Server xử lý và trả kết quả.

## Vai trò
- Client: giao diện người dùng
- Server: xử lý logic

## Java áp dụng
Java thường dùng mô hình này cho:
- Ứng dụng quản lý
- Website backend
- Hệ thống mạng nội bộ
