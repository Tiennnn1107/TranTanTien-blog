## Java là gì?
Java là một ngôn ngữ lập trình hướng đối tượng, được phát triển bởi Sun Microsystems.
Java nổi bật với khả năng **chạy trên nhiều nền tảng** nhờ cơ chế JVM.

## Vì sao Java phù hợp với lập trình mạng?
- Hỗ trợ thư viện mạng mạnh (`java.net`)
- Chạy ổn định trên server
- Bảo mật tốt
- Được dùng nhiều trong hệ thống Client–Server

## Ứng dụng thực tế
- Server backend
- Ứng dụng chat
- Hệ thống phân tán
