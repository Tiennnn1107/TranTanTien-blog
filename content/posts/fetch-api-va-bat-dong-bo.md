---
title: "Fetch API và xử lý bất đồng bộ trong JavaScript"
date: 2025-12-27
tags: ["javascript", "fetch", "api"]
---

Fetch API là công cụ mạnh mẽ cho phép JavaScript gửi và nhận dữ liệu từ Server. Nó thường được sử dụng để gọi REST API trong các ứng dụng web.

Fetch hoạt động theo cơ chế bất đồng bộ, giúp website không bị đứng khi chờ dữ liệu từ Server. Điều này giúp cải thiện trải nghiệm người dùng.

Qua việc sử dụng Fetch API, mình hiểu rõ hơn cách Client giao tiếp với Server và cách xử lý dữ liệu trả về một cách hiệu quả.
