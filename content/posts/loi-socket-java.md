## Một số lỗi phổ biến
- Port đã được sử dụng
- Không kết nối được server
- Đọc dữ liệu bị treo

## Nguyên nhân
- Server chưa chạy
- Firewall chặn
- Sai IP hoặc port

## Cách khắc phục
- Kiểm tra port
- Bắt exception
- Log lỗi rõ ràng
